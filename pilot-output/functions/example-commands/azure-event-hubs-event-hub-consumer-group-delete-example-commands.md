### Example CLI commands

Basic usage:

```azurecli
azmcp eventhubs eventhub consumergroup delete
```

With parameters:

```azurecli
azmcp eventhubs eventhub consumergroup delete --resource-group <resource-group> --namespace <namespace> --eventhub <eventhub> --consumer-group <consumer-group>
```
