### Example CLI commands

Basic usage:

```azurecli
azmcp sql server create
```

With parameters:

```azurecli
azmcp sql server create --resource-group <resource-group> --server <server> --administrator-login <administrator-login> --administrator-password <administrator-password> --location <location> --version <version> --public-network-access <public-network-access>
```
