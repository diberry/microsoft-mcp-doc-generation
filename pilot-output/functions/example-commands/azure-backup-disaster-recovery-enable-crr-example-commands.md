### Example CLI commands

Basic usage:

```azurecli
azmcp azurebackup disasterrecovery enable-crr
```

With parameters:

```azurecli
azmcp azurebackup disasterrecovery enable-crr --resource-group <resource-group> --vault <vault> --vault-type <vault-type>
```
