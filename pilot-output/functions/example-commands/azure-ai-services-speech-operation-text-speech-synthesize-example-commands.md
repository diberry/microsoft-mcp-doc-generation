### Example CLI commands

Basic usage:

```azurecli
azmcp speech tts synthesize
```

With parameters:

```azurecli
azmcp speech tts synthesize --endpoint <endpoint> --text <text> --outputAudio <outputAudio> --language <language> --voice <voice> --format <format> --endpointId <endpointId>
```
