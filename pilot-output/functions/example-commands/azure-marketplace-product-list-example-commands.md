### Example CLI commands

Basic usage:

```azurecli
azmcp marketplace product list
```

With parameters:

```azurecli
azmcp marketplace product list --language <language> --search <search> --filter <filter> --orderby <orderby> --select <select> --next-cursor <next-cursor> --expand <expand>
```
