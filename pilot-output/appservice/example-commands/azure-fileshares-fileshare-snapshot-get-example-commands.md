### Example CLI commands

Basic usage:

```azurecli
azmcp fileshares fileshare snapshot get
```

With parameters:

```azurecli
azmcp fileshares fileshare snapshot get --resource-group <resource-group> --file-share-name <file-share-name> --snapshot-name <snapshot-name>
```
