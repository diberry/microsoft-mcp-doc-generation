### Example CLI commands

Basic usage:

```azurecli
azmcp storagesync service update
```

With parameters:

```azurecli
azmcp storagesync service update --resource-group <resource-group> --name <name> --incoming-traffic-policy <incoming-traffic-policy> --tags <tags> --identity-type <identity-type>
```
