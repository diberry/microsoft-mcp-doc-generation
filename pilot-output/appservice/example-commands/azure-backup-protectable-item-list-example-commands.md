### Example CLI commands

Basic usage:

```azurecli
azmcp azurebackup protectableitem list
```

With parameters:

```azurecli
azmcp azurebackup protectableitem list --resource-group <resource-group> --vault <vault> --vault-type <vault-type> --workload-type <workload-type> --container <container>
```
