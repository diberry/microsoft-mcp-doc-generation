### Example CLI commands

Basic usage:

```azurecli
azmcp monitor workspace log query
```

With parameters:

```azurecli
azmcp monitor workspace log query --resource-group <resource-group> --workspace <workspace> --table <table> --query <query> --hours <hours> --limit <limit>
```
