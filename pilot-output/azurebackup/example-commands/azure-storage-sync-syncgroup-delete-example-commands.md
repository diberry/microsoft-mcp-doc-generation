### Example CLI commands

Basic usage:

```azurecli
azmcp storagesync syncgroup delete
```

With parameters:

```azurecli
azmcp storagesync syncgroup delete --resource-group <resource-group> --name <name> --sync-group-name <sync-group-name>
```
