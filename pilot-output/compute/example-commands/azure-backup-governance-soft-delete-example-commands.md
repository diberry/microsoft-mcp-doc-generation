### Example CLI commands

Basic usage:

```azurecli
azmcp azurebackup governance soft-delete
```

With parameters:

```azurecli
azmcp azurebackup governance soft-delete --resource-group <resource-group> --vault <vault> --vault-type <vault-type> --soft-delete <soft-delete> --soft-delete-retention-days <soft-delete-retention-days>
```
