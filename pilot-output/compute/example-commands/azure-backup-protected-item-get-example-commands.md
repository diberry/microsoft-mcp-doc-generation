### Example CLI commands

Basic usage:

```azurecli
azmcp azurebackup protecteditem get
```

With parameters:

```azurecli
azmcp azurebackup protecteditem get --resource-group <resource-group> --vault <vault> --vault-type <vault-type> --protected-item <protected-item> --container <container>
```
