### Example CLI commands

Basic usage:

```azurecli
azmcp storagesync cloudendpoint changedetection
```

With parameters:

```azurecli
azmcp storagesync cloudendpoint changedetection --resource-group <resource-group> --name <name> --sync-group-name <sync-group-name> --cloud-endpoint-name <cloud-endpoint-name> --directory-path <directory-path> --change-detection-mode <change-detection-mode> --paths <paths>
```
