### Example CLI commands

Basic usage:

```azurecli
azmcp azurebackup governance immutability
```

With parameters:

```azurecli
azmcp azurebackup governance immutability --resource-group <resource-group> --vault <vault> --vault-type <vault-type> --immutability-state <immutability-state>
```
