### Example CLI commands

Basic usage:

```azurecli
azmcp virtualdesktop hostpool host user-list
```

With parameters:

```azurecli
azmcp virtualdesktop hostpool host user-list --resource-group <resource-group> --hostpool <hostpool> --hostpool-resource-id <hostpool-resource-id> --sessionhost <sessionhost>
```
