### Example CLI commands

Basic usage:

```azurecli
azmcp managedlustre fs blob autoimport delete
```

With parameters:

```azurecli
azmcp managedlustre fs blob autoimport delete --resource-group <resource-group> --filesystem-name <filesystem-name> --job-name <job-name>
```
