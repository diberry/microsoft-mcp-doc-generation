### Example CLI commands

Basic usage:

```azurecli
azmcp eventhubs eventhub delete
```

With parameters:

```azurecli
azmcp eventhubs eventhub delete --resource-group <resource-group> --namespace <namespace> --eventhub <eventhub>
```
