### Example CLI commands

Basic usage:

```azurecli
azmcp speech stt recognize
```

With parameters:

```azurecli
azmcp speech stt recognize --endpoint <endpoint> --file <file> --language <language> --phrases <phrases> --format <format> --profanity <profanity>
```
