### Example CLI commands

Basic usage:

```azurecli
azmcp loadtesting testrun get
```

With parameters:

```azurecli
azmcp loadtesting testrun get --test-resource-name <test-resource-name> --resource-group <resource-group> --testrun-id <testrun-id> --test-id <test-id>
```
