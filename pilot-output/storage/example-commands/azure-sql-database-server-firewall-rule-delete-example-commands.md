### Example CLI commands

Basic usage:

```azurecli
azmcp sql server firewall-rule delete
```

With parameters:

```azurecli
azmcp sql server firewall-rule delete --resource-group <resource-group> --server <server> --firewall-rule-name <firewall-rule-name>
```
