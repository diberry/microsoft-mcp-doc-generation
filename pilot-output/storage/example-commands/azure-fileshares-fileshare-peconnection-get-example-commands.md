### Example CLI commands

Basic usage:

```azurecli
azmcp fileshares fileshare peconnection get
```

With parameters:

```azurecli
azmcp fileshares fileshare peconnection get --resource-group <resource-group> --file-share-name <file-share-name> --connection-name <connection-name>
```
