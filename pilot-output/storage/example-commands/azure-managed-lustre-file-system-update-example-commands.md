### Example CLI commands

Basic usage:

```azurecli
azmcp managedlustre fs update
```

With parameters:

```azurecli
azmcp managedlustre fs update --resource-group <resource-group> --name <name> --maintenance-day <maintenance-day> --maintenance-time <maintenance-time> --no-squash-nid-list <no-squash-nid-list> --squash-uid <squash-uid> --squash-gid <squash-gid> --root-squash-mode <root-squash-mode>
```
