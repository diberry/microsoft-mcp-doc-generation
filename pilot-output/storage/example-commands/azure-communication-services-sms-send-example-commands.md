### Example CLI commands

Basic usage:

```azurecli
azmcp communication sms send
```

With parameters:

```azurecli
azmcp communication sms send --endpoint <endpoint> --from <from> --to <to> --message <message> --enable-delivery-report <enable-delivery-report> --tag <tag>
```
